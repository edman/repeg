
luarocks
- install luarocks (it's in the arch linux community repository)
	pacman -Syu luarocks
- download the lpeg rock
	luarocks install lpeg
	- if you have any problem with permissions, install it locally
		luarocks --local install lpeg
- make lua know about the luarocks loader library by calling the library
when invoking the lua interpreter (this way you can use the rocks just as
any other lua module)
	lua -lluarocks.loader
	- you may wish to make this an alias to lua in your .bashrc
		alias lua -lluarocks.loader

test.lua
- to execute tests you must execute this source with the lua interpreter
giving the path for a second file containing the test information
	lua test.lua samples/sample-original.lua
